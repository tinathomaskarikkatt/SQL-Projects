# 🗄️ Workforce Income Analysis using SQL Server

A SQL-based workforce analytics project that explores global salary trends, workforce distribution, and remote work patterns using SQL Server. The project analyzes workforce data from 2020–2024 to generate business insights that support salary benchmarking, hiring strategies, and workforce planning.

---

# 📌 Project Overview

Organizations need workforce analytics to make informed decisions regarding compensation, recruitment, and remote work policies. This project analyzes a global workforce dataset containing salary, job role, experience level, employment type, company size, and location data using SQL Server.

The analysis answers key business questions related to workforce distribution, salary growth, geographic pay differences, and remote work adoption while demonstrating practical SQL techniques for data analysis.

---

# 🎯 Project Objectives

- Analyze salary trends across job titles and experience levels.
- Compare salaries across countries and company sizes.
- Study remote work adoption between 2020 and 2024.
- Identify high-paying job roles and locations.
- Support workforce planning and salary benchmarking through data-driven insights.

---

# 📊 Business Questions Explored

The SQL queries answer several real-world business questions, including:

- Which countries offer the highest average salaries?
- How do salaries vary across experience levels?
- Which company sizes employ the most professionals?
- Which employment type is most common?
- How has remote work changed over time?
- Which job titles receive the highest salaries?
- Which countries are ideal for workforce expansion?
- Which career transitions result in higher salaries?

---

# 📈 Key Findings

- Large companies employ the majority of professionals compared to medium and small organizations.
- Full-time employment dominates across most job roles.
- The United States, India, Germany, Canada, and the United Kingdom have the largest concentration of large companies.
- Remote work adoption declined steadily between 2021 and 2024.
- Approximately **75% of fully remote employees earn more than $100,000 annually.**
- Mid-level professionals experienced the strongest salary growth between 2023 and 2024.
- Significant salary differences exist across countries, highlighting the importance of regional salary benchmarking.
- Internal role transitions can result in substantial salary increases for professionals.
